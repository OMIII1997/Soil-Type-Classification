कछार की मिट्टी में आप यह फसलं ले सकते है ।:- { चावल, गेहूं, गन्ना, मक्का, कपास, सोयाबीन, जूट }


કાંપવાળી જમીનમાં તમે આ પાક લો છો ।:- { ચોખા, ઘઉં, શેરડી, મકાઈ, કપાસ, સોયાબીન, જૂટ}


வண்டல் மண்ணில் நீங்கள் இந்த பயிர்களை எடுத்துக்கொள்கிறீர்கள் ।:- { அரிசி, கோதுமை, கரும்பு, மக்காச்சோளம், பருத்தி, சோயாபீன், சணல் }
