Soil Type : Black Soil

You can Plant below crops:- { Virginia, Wheat , Jowar,Millets,Linseed,Castor,Sunflower }


काळ्या माती मध्ये तुम्ही ही पिके घेऊ शकता:- { व्हर्जिनिया, गहू, ज्वारी, मिलेट्स, बिसीड, एरंडेल, सूर्यफूल }


काली मिटटी में आप यह फसलं ले सकते है ।:- { वर्जीनिया, गेहूं, ज्वार, बाजरा, अलसी, अरंडी, सूरजमुखी }


કાળી જમીનમાં તમે આ પાક લઈ શકો છો ।:- { વર્જિનિયા, ઘઉં, જુવાર, મિલેટ્સ, અળસી, એરંડા, સૂર્યમુખી }


கருப்பு மண்ணில் நீங்கள் இந்த பயிர்களை எடுக்கலாம் ।:- { வர்ஜீனியா, கோதுமை, ஜோவர், தினை, லின்சீட், ஆமணக்கு, சூரியகாந்தி }


ಕಪ್ಪು ಮಣ್ಣಿನಲ್ಲಿ ನೀವು ಈ ಬೆಳೆಗಳನ್ನು ತೆಗೆದುಕೊಳ್ಳಬಹುದು ।: -{ ವರ್ಜೀನಿಯಾ, ಗೋಧಿ, ಜೋವರ್, ರಾಗಿ, ಲಿನ್‌ಸೀಡ್, ಕ್ಯಾಸ್ಟರ್, ಸೂರ್ಯಕಾಂತಿ }
