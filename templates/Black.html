Soil Type : Black Soil

You can Plant below crops:- { Jowar,Millets,Linseed,Castor,Sunflower }


काळ्या माती मध्ये तुम्ही ही पिके घेऊ शकता:- { ज्वार, मिल्ट्स, अलसी, एरंडेल, सूर्यफूल }


काली मिटटी में आप यह फसलं ले सकते है ।:- { ज्वार, बाजरा, अलसी, केस्टर, सूरजमुखी }


કાળી જમીનમાં તમે આ પાક લઈ શકો છો ।:- { જુવાર, મિલેટ્સ, અળસી, એરંડા, સૂર્યમુખી }


கருப்பு மண்ணில் நீங்கள் இந்த பயிர்களை எடுக்கலாம் ।:- { ஜோவர், தினை, லின்சீட், ஆமணக்கு, சூரியகாந்தி }


ಕಪ್ಪು ಮಣ್ಣಿನಲ್ಲಿ ನೀವು ಈ ಬೆಳೆಗಳನ್ನು ತೆಗೆದುಕೊಳ್ಳಬಹುದು ।: -{ ಜೋವರ್, ರಾಗಿ, ಲಿನ್‌ಸೀಡ್, ಕ್ಯಾಸ್ಟರ್, ಸೂರ್ಯಕಾಂತಿ }
